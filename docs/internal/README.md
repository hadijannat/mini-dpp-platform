# Internal Documentation

Internal docs cover audits, reviews, evidence, and planning artifacts.

## Sections

- `audits/`: inspection outputs and alignment matrices
- `reviews/`: review reports and expert assessments
- `plans/`: implementation and remediation plans
- `evidence/`: collected evidence and progress logs
- `agent-teams/`: agent operating playbooks
- `claims/`: claim-to-evidence mapping
- `meta/`: documentation standards and templates
